# Basic Express App

A simple Node.js application using Express that responds with `"Hello"` when accessed via HTTP GET request on the root route (`/`).

## Features

- ✅ Uses Express.js
- ✅ Handles GET request on `/`
- ✅ Returns JSON response: `"Hello"`
